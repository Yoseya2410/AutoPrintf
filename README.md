# AutoPrintf

这是一个自动打字工具，可以绕过禁止粘贴限制的自动输入工具，能够模拟人工打字将剪贴板中的文本内容输入到任何输入框中

## 功能特点

- 绕过禁止粘贴的输入框限制
- 模拟键盘输入文本内容
- 支持中途停止输入

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 将需要输入的文本内容复制到剪贴板
2. 运行程序：
   ```bash
   python auto_typewriter.py
   ```
3. 将光标放到想输入的输入框内
4. 按下 `Ctrl+Alt+V` 开始自动输入
5. 输入过程中按 `Esc` 键可随时停止

## 注意事项

- 本工具仅能处理纯文本内容
- 请确保输入法为英文模式，否则可能出现输入错误
- 在输入过程中请勿移动鼠标或操作键盘，以免影响输入效果
- 如需紧急停止，可多次按 `Esc` 键或直接关闭程序窗口
